# Rightview MVP

A mobile app that lets users interact with a map of properties in a given location through a conversational interface.

## Overview

- Interactive map with property pins
- Conversational AI interface to query and explore properties
- Built for the Indian real estate market

## MCP Servers Used

- `@modelcontextprotocol/server-google-maps` — Geocoding, place search, map data
- `@modelcontextprotocol/server-memory` — Persistent session memory
- `mcp-server-fetch` — Web content fetching
- Claude built-in web search — Neighborhood and area enrichment

## Project Structure
